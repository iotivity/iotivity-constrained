Steps to create the doxygen documents:
1. Go to "docs" directory.
    $ cd ${IOTIVITY_BASE}/docs
2. Generate the doxygen html documents.
    $ doxygen iotivity_lite.doxyfile
    $ doxygen iotivity_st_app_fw_api.doxyfile

The generated documents are present in following diretories: 
    ${IOTIVITY_BASE}/docs/iotivity_lite
    ${IOTIVITY_BASE}/docs/iotivity_st_app_fw_api
